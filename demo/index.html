Tailwind Breakout Grid

A flexible, fluid grid layout system for content that breaks out of its container. Press Cmd+G to toggle the visualizer.

Basic Column Widths

Each column width serves a different purpose. Narrow is for readable text, content for general layout, and wider columns for emphasis.

.col-full Edge to edge
.col-full-limit Full width with max-width constraint
.col-feature Extra wide for heroes & images
.col-popout Slightly wider than content
.col-content Standard content width (default)
.col-narrow Optimal reading width (40-50rem)

Padding Utilities

Use px-breakout on full-width sections to align inner content with the content column.

.col-full .px-breakout

This content aligns with .col-content edges thanks to px-breakout padding.

.col-full .px-gap

Uses the responsive grid gap as padding.

.col-popout .px-breakout

Popout width with breakout padding for inner alignment.

Left & Right Alignment

Span from one edge to a specific column boundary for asymmetric layouts.

.col-feature-left Left edge to feature boundary
Feature boundary to right edge .col-feature-right
.col-center-left Left edge to center point
Center point to right edge .col-center-right

CSS Subgrid

Use .grid-cols-breakout-subgrid on nested elements to inherit the parent grid's tracks. Children can align to any named line.

.col-feature - spans the feature area
.col-narrow - aligns to narrow!
.col-content - aligns to content!

Parent: .col-feature-right .grid-cols-breakout-subgrid

.col-popout - popout width
.col-content - content alignment

Parent: .col-feature-left .grid-cols-breakout-subgrid

Browser Support: CSS Subgrid has ~90% browser support as of January 2025. Check current support

Nested Grids

Use breakout-to-* modifiers when nesting grids inside constrained containers.

Parent: .col-feature

.col-feature fills the container
.col-content has proper margins
.col-narrow for readable text

Nested: .grid-cols-breakout .breakout-to-feature

Column Width Hierarchy

From narrowest to widest - resize your browser to see responsive behavior

col-narrow

Optimal reading width (40-50rem). Perfect for long-form text content and articles.

col-content (default)

Standard content width (~4vw on each side). This is the default column if no class is specified.

col-popout

Pops out slightly into the margins (~5rem on each side). Great for blockquotes and callouts.

col-feature

Extra wide for emphasis content (~6vw extension). Ideal for hero images and featured content.

col-feature-popout

Maximum content width (~5rem + 6vw). Perfect for hero sections and major visual emphasis.

col-full

Full viewport width (edge-to-edge with gap padding). Use for full-bleed sections.

col-full-limit

Full width but never exceeds fullLimit (120rem by default). Best for wide containers on ultra-wide screens.

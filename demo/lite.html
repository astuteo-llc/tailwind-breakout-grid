Standalone CSS Demo

This demo uses the standalone CSS export—no Tailwind plugin required for the grid! Press Cmd+G to toggle the lite visualizer.

The lite visualizer shows grid overlay and measurements without config editing.

How It Works

The standalone CSS provides all breakout grid utilities without the Tailwind plugin.

Standalone CSS (inline or external file) 
<!-- Option 1: Link to exported CSS file -->
<link rel="stylesheet" href="breakout-grid.css">

<!-- Option 2: Inline the CSS (see this page's source) -->
<style>
  :root {
    --base-gap: 1rem;
    --content-min: 53rem;
    /* ... full CSS from visualizer export */
  }
  .grid-cols-breakout { ... }
  .col-content { grid-column: content; }
  /* etc. */
</style>

<!-- Use the grid exactly like the Tailwind version -->
<div class="grid-cols-breakout">
  <p class="col-content">Reading width</p>
  <img class="col-feature" src="wide.jpg">
</div>

Column Widths

These work identically to the Tailwind plugin version.

.col-full Edge to edge
.col-full-limit Full width with max-width constraint
.col-feature Extra wide for heroes & images
.col-popout Slightly wider than content
.col-content Optimal reading width

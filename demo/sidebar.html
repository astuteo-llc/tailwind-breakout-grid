Sidebar Layout Pattern

This page demonstrates how to use a sidebar within the breakout grid system. The key classes:

<main class="grid-cols-breakout">
<div class="col-feature breakout-none-grid grid-cols-12 gap-8">
<aside class="col-span-3">Sidebar</aside>
<div class="col-span-9">
<!-- Blocks with col-* classes render normally -->
<div class="col-popout">Ignored!</div>
</div>
</div>
</main>
grid-cols-breakout

Outer breakout grid (page-level)

col-feature

Container width for sidebar area

breakout-none-grid

Disables breakout, enables CSS grid

Shared Components: The blocks below are loaded from shared-blocks.js — the exact same source used on the Product page. Compare how the col-* classes work there vs. here.
